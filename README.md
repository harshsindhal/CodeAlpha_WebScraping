# Web Scraping and Analysis of Book Data Using Python

## Project Overview

This project demonstrates the use of web scraping techniques to collect and analyze book data from a public website. Using Python, BeautifulSoup, and Pandas, data was extracted, cleaned, and transformed into a structured dataset for analysis and visualization.

The objective of this project is to understand the process of automated data collection and derive meaningful insights from the scraped data.

## Dataset Source

Website: Books to Scrape

The dataset was created by scraping book information directly from the website and storing it in a CSV file.

## Tools and Technologies

* Python
* Requests
* BeautifulSoup
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Research Questions

1. What is the distribution of book ratings?
2. What is the price distribution of books?
3. Which rating category appears most frequently?
4. Are most books available in stock?

## Data Collected

The following information was extracted:

* Book Title
* Price
* Rating
* Availability Status

## Analysis Performed

* Web scraping and data extraction
* Data cleaning and preprocessing
* Dataset creation and export to CSV
* Rating distribution analysis
* Price distribution analysis
* Data visualization

## Visualizations

### Books by Rating

Displays the distribution of books across different rating categories.

### Price Distribution

## Key Insights

* Successfully extracted and structured book information from a live website.
* Most books fall within a moderate price range.
* Rating distribution provides insights into the popularity and quality of books.
* Web scraping can automate data collection for analytical purposes.

## Conclusion

This project demonstrates how web scraping can be used to collect real-world data and transform it into actionable insights. By combining Python, BeautifulSoup, and data analysis libraries, raw web data can be converted into meaningful datasets suitable for business intelligence and decision-making.
